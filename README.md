Expense Tracker – Data Analysis Project

Project Overview

This Expense Tracker project focuses on analyzing personal financial data to understand income, expenses, and savings. The objective is to transform raw transaction data into meaningful insights that support better financial planning and spending control.

Project Objectives

- Track income and expenses accurately  
- Categorize transactions for better analysis  
- Analyze monthly spending trends  
- Identify high-expense categories  
- Evaluate savings and spending behavior  

Dataset Description

The dataset contains personal transaction details, including:

- Date  
- Category and sub-category  
- Income and expense amounts  
- Payment mode  
- Month and year  

Data cleaning and preprocessing were performed before analysis.

Tools & Technologies Used

- Excel – Data cleaning, transformation, and pivot analysis  
- Power BI – Data modeling, DAX measures, and interactive dashboards  

Data Modeling

A structured data model was used:

- Fact Table: Amount, Income, Expense, Net Balance  
- Dimension Tables: Date, Category, Payment Mode  

This structure improves performance and analytical clarity.

Analysis Performed

- Descriptive analysis of income, expenses, and savings  
- Category-wise expense distribution  
- Monthly expense trend analysis  
- Spending vs savings comparison  

Key Insights

- Certain categories contribute the highest expenses  
- Monthly spending patterns vary over time  
- Better expense control improves savings  
- Digital payment methods are widely used  

Dashboard Features

- KPI cards for total income, total expenses, and savings  
- Category-wise expense charts  
- Monthly trend analysis visuals  
- Interactive slicers for month and category  

Conclusion

This project demonstrates end-to-end data analysis skills, including data cleaning, modeling, visualization, and insight generation. It is suitable for showcasing Excel and Power BI skills in a data analyst portfolio.
